In the database there are records of:
1) Non-Electronic Objects (Physical Objects)
2) Heightmaps
3) 3D objects
4) Images
5) Publications
6) Links to External Resources
organized in different folders for easy management.

Each record consists of a list of entries (fields/values) and a list of links to other records. Examples:
<field>
	<field_id>1</field_id>
	<field_value>5844ac196e2970c6</field_value>
</field>
<record_link>
	<neobject_id>05e2729b149912bf</neobject_id>
</record_link>
<record_link>
	<heightmap_id>05e2729492bf</heightmap_id>
</record_link>

Each field may optionally have a list of sources from which the information was taken:
<source>
	<url>http://www.metmuseum.org/Collections/search-the-collections/130008463</url>
</source>
<source>
	<citation_id>8463</citation_id>
</source>
